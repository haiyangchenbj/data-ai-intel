来源： GitHub dbt-labs/dbt-core

来源： Databricks Release Notes
摘要： Databricks 于 6 月 17 日上线 Genie Space 外部共享功能（Beta），通过 OpenSharing 协议向组织外部用户共享 Genie Space 的数据资产和指令快照，接收方可挂载共享创建本地 Genie Space。
对数据平台的影响： 将 AI Agent 知识空间纳入开放共享协议，标志着数据共享从表级数据扩展到 Agent 上下文层，这对企业间 AI 协作的治理框架提出了新要求——如何在跨组织 Agent 环境中维持一致的语义与权限模型。
