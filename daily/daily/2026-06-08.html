Top Signals

Databricks 超 10 亿美元收购 Tabular，Iceberg 创始人团队加入意图统一开放表格式标准

来源：TechTarget、Databricks 官方博客、CNBC
摘要： Databricks 于 6 月 4 日宣布以超 10 亿美元收购数据管理公司 Tabular，后者由 Apache Iceberg 项目创始人 Ryan Blue 和 Daniel Weeks 于 2021 年创立，累计融资 3700 万美元。收购完成后，Tabular 三位联合创始人将加入 Databricks，与 Databricks 共同承诺推动 Delta Lake 与 Iceberg 向统一开放标准演进。Databricks 同时表示将继续通过 UniForm（6 月 3 日 GA）维护 Delta Lake、Iceberg 和 Hudi 的三向兼容性。该交易预计于 Databricks 财年 Q2（7 月 31 日前）完成。
3点：
- 格式侧，Iceberg 核心创始人加入并承诺统一标准，意味着湖仓两大主流格式将从竞争走向融合，企业不再承担格式锁定风险
- 生态侧，Databricks 此前仅在 Delta Lake 社区有深度影响力，收购后在 Iceberg 社区也将获得同等话语权，对主要竞争对手构成直接压力
- 架构侧，统一开放表格式是 AI Agent 跨系统访问数据的前提条件，此举为 Agent 可自由遍历的数据层扫清格式障碍
So What： 开放表格式的统一比任何单一产品发布都更具结构性意义——它消除了 Agent 时代数据互操作的最大摩擦点，也让"湖仓一体"从营销概念变为真正的工程现实。Snowflake 在 Summit 期间力推的 Iceberg 互操作战略，此刻面临 Databricks"以收购代竞争"的降维回应。

Databricks 双发 AI Search 品牌重塑与 Lakebase 统一治理，定义 Agent-Native 数据平台

来源：Databricks 产品文档、Databricks AI Search 产品页、The Agent Times
摘要： Databricks 在 6 月 5 日同步推进两项重大产品升级：（1）Vector Search 正式更名为 AI Search，新增无需向量/嵌入的纯全文搜索索引能力，支持语义、关键字（BM25）和混合搜索统一 API；（2）Lakebase 正式发布，将运营数据库（operational databases）纳入 Unity Catalog 统一治理层，消除事务型系统与分析型系统之间的治理割裂，并演示了与 Spotify Backstage 开源开发者门户的集成。Databricks 6 月更新还包括：Managed SharePoint connector 支持结构化文件摄入（Beta）、Excel Add-in 支持回写数据到 Unity Catalog、Genie 预算管控、Full page Genie Code（Beta）等。
3点：
- 搜索侧，AI Search 从纯向量检索扩展为混合语义与关键字搜索，意味着 RAG 和 Agent 应用不再需要拼接多个搜索系统
- 治理侧，Lakebase 将运营数据库纳入统一治理层，使 Agent 在事务系统和分析系统之间穿梭时无需切换权限模型
- 产品侧，搜索升级、治理统一与 Tabular 收购形成完整叙事，正在构建 Agent 可以自主发现、访问、分析和操作数据的闭环环境
So What： Databricks 本周的动作组合表明，数据平台竞争的下一个前沿不再是"谁能存储更多数据"，而是"谁能让 Agent 更高效地使用数据"。AI Search 解决"找得到"，Lakebase 解决"管得统一"，Tabular 解决"格式不卡"——三者叠加构成 Agent-Native 数据基础设施的完整拼图。

腾讯云面向 Agent 升级全栈数据平台：DataBuddy + WeData + AI 原生底座三层架构

来源：腾讯新闻、科创板日报、搜狐科技
摘要： 6 月 5 日，腾讯云在 AI 产业应用大会上宣布面向 Agent 升级全栈数据平台能力，构建三层架构：（1）智能入口 DataBuddy——生产级数据智能体，支持自然语言驱动数据建模、ETL 开发、任务编排、归因分析和报告生成，内测以来超 3000 家企业申请，可将重复开发工作量降低 80%；（2）统一控制面 WeData——一站式数据智能平台，提供统一元数据、语义层、编排和权限体系，自然语言转 SQL 准确率提升至 90% 以上；（3）AI 原生数据底座——从存储、计算引擎、系统到数据分析四层重构，自研 Meson 引擎 TPC-DS 性能提升 3.6 倍，Xpark 跨模态引擎推理吞吐提升 3 倍。同步升级 TBDS 私有化大数据平台面向政务金融场景。
3点：
- 用户侧，腾讯云明确将 Agent 定位为与人的并列使用者，平台设计从人操作工具转向人与 Agent 协同完成任务
- 语义侧，统一语义层将指标口径、业务术语沉淀为 Agent 可理解的资产，解决 Agent 调用数据时的理解断层问题
- 底座侧，存储、计算、数据与系统四层全面重构，表明这不是功能叠加而是面向 Agent 时代的范式跃迁
So What： 腾讯云此次升级与 Databricks 本周动作形成中美市场的罕见共振——双方都意识到 Agent 不仅需要"能访问数据"，更需要"能理解数据语义"和"能在统一治理下操作数据"。WeData 的统一语义层和 Databricks 的 Lakebase 统一治理，分别从"语义理解"和"权限治理"两个维度回应同一命题。Agent-Native 数据基础设施的全球竞赛已正式开赛。

Views & Research

Constellation Research：Databricks 收购 Tabular 是"Snowflake Summit 聚光灯下的正面交锋"

来源：TechTarget
摘要： Constellation Research 分析师 Doug Henschen 评论称："The timing of this deal is obviously intended to grab some of the Snowflake Summit limelight but also to outdo its competitor on openness messaging with the suggestion that it will have huge influence over the future of the Iceberg standard." 他指出，Databricks 此前已通过 UniForm 实现 Delta Lake/Iceberg/Hudi 三向兼容，因此收购并不令人意外，但真正的意义在于"The deal brings Databricks depth and influence in the Iceberg community, whereas it previously only had depth and obvious influence over Delta Lake"。

Smart Chunks：Databricks AI Search 品牌重塑标志着"混合搜索成为准入门槛"

来源：Smart Chunks
摘要： Smart Chunks 分析指出，Databricks 更名 AI Search 反映了一个工程现实：生产级 AI Agent 和 Copilot 不生活在纯向量空间，它们需要"find me contracts mentioning ACME Corp signed after Q3 2025 that are semantically similar to this clause"这类混合查询。文章认为向量搜索是"为单一路况设计的跑车"，而 AI Search 是"能同时应对高速、城市和土路的跨界车"——因为真实检索工作流会在同一查询中混合所有三种需求。

The Agent Times：Lakebase 消除了 Agent 跨治理边界操作的"摩擦乘数"

来源：The Agent Times
摘要： The Agent Times 分析指出，AI Agent 当前需要在运营数据库读取、分析仓库查询、事务系统回写之间穿梭，每次跨越都涉及独立的凭证链、API 接口和模式表示。Lakebase 通过 Unity Catalog 提供统一的权限和发现层，Agent 可在不切换治理上下文的情况下完成全链路操作。文章称"fragmented data governance is a friction multiplier"——每一个额外的权限边界都会增加延迟、故障模式和安全暴露面。
