AWS EMR Serverless 支持 32 vCPU / 244 GB 大规格 Worker

来源： AWS What's New
摘要： AWS EMR Serverless 新增 32 vCPU / 最高 244 GB 内存的大规格 Worker 配置，此前最大配置为 16 vCPU / 120 GB。大规格 Worker 可减少 Shuffle 密集型作业的跨节点数据传输、降低数据倾斜场景下的 OOM 风险、以及支持更大规模的内存缓存。
对数据平台的影响： 无服务器 Spark/Hive 处理能力向上扩展，使 EMR Serverless 可以承接原本需要自建集群的高内存负载（如大规模 join、宽表聚合），进一步模糊了 Serverless 与自建集群之间的能力边界。
