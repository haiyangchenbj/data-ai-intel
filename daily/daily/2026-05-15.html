Top Signals

1. Snowflake Cortex AI Guardrails 扩展至 Intelligence 和 Cortex Agents（GA）

来源：Snowflake Release Notes 2026-05-14

Snowflake 将 Cortex AI Guardrails 的覆盖范围从 Cortex Code 扩展至 Snowflake Intelligence 和 Cortex Agents，正式达到 GA 状态。该功能使用上下文推理检测提示注入和越狱攻击，并能识别嵌入在工具调用中的间接提示注入。管理员可通过单一 AI_SETTINGS 参数为所有 AI 服务统一启用防护。

为什么对数据平台重要：这是 Snowflake 首次在数据平台层面提供覆盖全部 AI 服务的统一安全防护。当 Agent 和 AI 函数直接操作企业数据时，提示注入可能导致数据泄露或错误操作——将安全防护嵌入平台运行时（而非留给应用层自行处理）正在成为数据平台的新竞争维度。

2. LangChain Interrupt 2026 完整发布总结：Agent 基础设施全栈升级

来源：LangChain Blog - Everything we shipped at Interrupt

LangChain 在 Interrupt 2026 大会（5/13-14）集中发布了 10 项产品更新，Day 2（5/14）公布完整发布总结和 LangChain Labs 新实验室。两日发布涵盖：SmithDB（Agent 可观测性数据库）、LangSmith Engine、Context Hub、Sandboxes GA、LLM Gateway（运行时治理）、Deep Agents v0.6 及 Managed Deep Agents、Delta Channels（长会话增量检查点，存储降低 41 倍）。

为什么对数据平台重要：这是目前 Agent 生态最密集的基础设施发布窗口。Delta Channels 解决了长运行 Agent 的 O(N²) 检查点存储爆炸问题（500 轮对话从 4GB 降至 110MB），LLM Gateway 引入运行时治理层——这些都是 Agent 从原型走向生产的关键基础设施，数据平台厂商在考虑 Agent 集成时需要关注这些新的存储和治理模式。

Product & Tech

1. Databricks Lakeflow Spark 声明式管道支持环境版本锁定（Beta）

来源：Databricks Release Notes May 2026

Databricks 为 Lakeflow Spark Declarative Pipelines（原 DLT）新增环境版本功能（Beta），允许用户在管道上配置环境版本以锁定 Python 语言版本和预装库集合，使管道的 Python 运行时与 Databricks Runtime 升级解耦。

对数据平台的影响：解决了 Spark 管道升级的核心痛点——Runtime 版本升级导致的 Python 兼容性问题。环境版本锁定让数据工程团队可以按自己的节奏升级依赖，降低了大规模管道的维护风险。

2. Databricks SQL Alerts GA + Lakeflow Jobs SQL Alert 任务升级至 Public Preview

来源：Databricks Release Notes May 2026

Databricks SQL Alerts 最新版达到 GA，新增 Markdown 编辑器支持自定义通知模板。同步地，Lakeflow Jobs 中的 SQL Alert 任务从 Beta 升级至 Public Preview，支持在作业流中评估 SQL 告警并将结果作为任务输出值，下游任务可据此分支执行。

对数据平台的影响：SQL 告警从独立功能演进为可编排的工作流节点，使数据质量监控和异常检测可以直接嵌入数据管道的编排逻辑中，而非作为外挂的监控层。

跟踪 LangChain Deep Agents v0.6 + Delta Channels：长运行 Agent 存储效率验证

来源：LangChain Blog - Delta Channels

Delta Channels 将 LangGraph 检查点从全量快照改为增量 delta 写入（周期性快照），500 轮对话存储从 4GB 降至 110MB（41 倍）。这个优化已在 Deep Agents v0.6 中默认启用，并在 LangGraph v1.2 中以 API 形式开放。对数据平台而言，Agent 数据的存储模式正在从"状态全量复制"走向"增量+定期快照"——与数据库领域的 WAL/增量检查点思路高度一致。

需要等待什么信号确认：生产环境中 Delta Channels 的实际性能和稳定性反馈。
