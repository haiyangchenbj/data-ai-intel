A. Top Signals

1. 腾讯云 ADP 智能体开发平台宣布模型涨价：混元 HY2.0 涨幅超 460%，第三方模型结束免费

📌 腾讯云官方公告 | 新浪财经（2026-03-11）

3月11日，腾讯云宣布自3月13日起，智能体开发平台（Tencent Cloud ADP）对部分模型计费策略进行调整。GLM 5、MiniMax 2.5、Kimi 2.5 三个第三方模型将结束免费公测，正式商用计费。自研混元系列涨幅显著：

  • HY2.0 Instruct 输入：0.0008 → 0.004505 元/千Token ↑ 463%
  • HY2.0 Instruct 输出：0.002 → 0.01113 元/千Token ↑ 456%
  • HY2.0 Think 输入：0.001 → 0.0053 元/千Token ↑ 430%
  • HY2.0 Think 输出：0.004 → 0.0212 元/千Token ↑ 430%
📊 数据平台影响：Agent 模式下单任务 Token 消耗较 Chatbot 提升 3 个数量级，叠加涨价，企业 Agent 应用的推理成本将大幅上升。直接驱动数据平台必须提供 Token 消耗监控、成本归因、FinOps 仪表盘等能力。涨价信号验证 Agent 驱动的算力消耗已从"免费试用"走向真实商业化。

2. 工信部启动"工业数据筑基行动"：面向 AI 赋能建设高质量行业数据集

📌 中国新闻网 / 工信部"工信微报" | 新华社 / 新浪财经（通知发布 2026-03-10，3月11日广泛报道）

工信部印发《关于启动工业数据筑基行动 开展面向人工智能赋能的高质量行业数据集建设先行先试的通知》（工信厅信发函〔2026〕64号）。目标：到2026年底，培育行业数据合作联合体，建设重点行业数据可信互联平台，打造四大资源库（数据资源库、数据技术攻关库、工业数据标准库、高质量行业数据集库）。聚焦钢铁、汽车等重点行业，依托四类主体推动先行先试。

📊 数据平台影响：国家层面首次以专项通知推动工业数据标准化和 AI 赋能数据集建设，直接创造了数据平台在数据采集（"采"）、数据集构建（"集"）、数据应用（"用"）三环节的大规模需求。能提供"数据可信互联平台"能力的厂商将直接受益。

3. 微信秘密研发 AI Agent 项目曝光：打通数百万小程序，2026年中灰度测试

📌 The Information / 品玩 | 新浪科技 | ZAKER（2026-03-11）

据 The Information 报道，腾讯微信团队正秘密研发一款高优先级 AI Agent 产品，由微信技术负责人周颢负责、直接向张小龙汇报，2025年上半年启动。Agent 将全面接入微信生态内数百万小程序，实现跨应用任务执行——用户通过自然语言即可调用打车、外卖、订票、缴费等服务。计划2026年中灰度测试，Q3 逐步放量。技术选型上并行测试智谱、DeepSeek、阿里等外部模型及自研小模型。马化腾3月11日凌晨在朋友圈转发腾讯"龙虾"产品矩阵，配文"还有一批产品陆续赶来"。

📊 数据平台影响：微信 14 亿月活 + 数百万小程序 + AI Agent = 数据交互量量级跃升。Agent 需实时调用小程序 API、管理用户会话历史、处理跨服务数据流转，对数据平台的实时处理能力、会话存储、隐私合规提出全新挑战。可能成为中国覆盖面最广的 Agent 产品。

B. Product & Tech

1. 百度智能云发布 DuClaw：OpenClaw "零部署"服务，首购 17.8 元/月

📌 百度智能云 / 人民网 | 东方财富网（2026-03-11）
百度智能云发布 DuClaw，将"一键部署"升级为"零部署"——用户无需选择镜像、配置服务器或填写 API Key，订阅即用。首购 17.8 元/月。预置百度搜索、百科等 Skills，支持 DeepSeek、Kimi K2.5、GLM-5、MiniMax M2.5 等主流模型。后续将支持企微、钉钉、飞书。
📊 影响判断：Agent 部署门槛从"会运维"降至"会订阅"，Agent-as-a-Service 进入零门槛阶段。Token 消耗将加速集中到云端，推理侧数据管道需支撑更大规模集中式调用。

2. 国家超算互联网向全体 OpenClaw 用户免费发放 1000 万 Tokens

📌 超算互联网 / 东方财富网 | DoNews（2026-03-11）
国家超算互联网宣布面向全体 OpenClaw 用户免费发放每人 1000 万 Tokens（限时 2 周）。同时公布 Token 续购价格：0.1 元/百万 Tokens，较市场均价有明显降幅。此前（3月9日）已打通飞书、企业微信。
📊 影响判断：国家级算力基础设施入场补贴 Agent Token 消耗，与腾讯云涨价形成鲜明对比。价格分化将加速 Agent 算力市场分层，对数据平台的多云/混合云部署能力提出更高要求。

3. CNCERT 发布 OpenClaw 安全风险提示：默认配置脆弱，系统控制权可被获取

📌 CNCERT / 东方财富网 | 和讯网（2026-03-10，3月11日广泛传播）
CNCERT 发布《关于 OpenClaw 安全应用的风险提示》，指出 OpenClaw 被授予较高系统权限，但默认安全配置极为脆弱。已曝出多个高中危漏洞：(1) 提示词注入——攻击者诱导 Agent 泄露系统密钥；(2) 误操作——Agent 错误理解指令可能删除核心数据；(3) 功能插件投毒。
📊 影响判断：Agent 安全问题上升到国家网络安全机构关注层面。数据平台需提供细粒度权限控制、审计日志、数据沙箱等安全能力来支撑 Agent 安全运行。

4. Databricks 发布 LogSentinel：用 LLM 驱动 PII 检测与治理

📌 Databricks 官方博客（2026-03-06）
Databricks 发布 LogSentinel 案例——展示如何利用 LLM 自动检测和治理日志中的个人身份信息（PII）。这是 Databricks 自身使用自家平台进行数据治理的"dogfooding"实践。
📊 影响判断：LLM 驱动的 PII 检测代表数据治理新范式——从规则匹配转向语义理解，显著提升非结构化数据中的 PII 检出率。数据平台将治理能力与 LLM 深度集成是必然趋势。

5. Databricks 发布 KARL Agent：基于强化学习的企业知识检索加速

📌 Databricks 官方博客（2026-03-05）
Databricks 发布 KARL（A Faster Agent for Enterprise Knowledge），通过自定义强化学习（RL）优化企业知识库的检索和回答速度，旨在解决大规模企业知识检索中 Agent 响应延迟高的问题。
📊 影响判断：RL 优化 Agent 检索是降低企业 AI 应用延迟和 Token 消耗的关键技术路径。Databricks 正从"数据平台提供 AI 能力"走向"AI 优化数据平台自身性能"。

D. Analyst Insights

1. OpenClaw 驱动国产大模型调用量暴增：阶跃星辰日调用量增长超 20 倍

📌 科创板日报 / 搜狐（2026-03-11）

基于 OpenRouter 统计，OpenClaw 日榜与月榜前三均为中国大模型：阶跃星辰 Step 3.5 Flash、Kimi K2.5、MiniMax M2.5。关键数据：

  • 阶跃星辰 Step 3.5 Flash 日调用量较30天前 增长超 20 倍
  • MiniMax M2 系列 2月日均 Token 消耗较去年12月 增长超 6 倍
  • 其中 Coding Plan Token 消耗 增长超 10 倍
📊 数据平台映射：国产大模型调用量暴增意味着推理侧数据基础设施面临极大压力——日志系统、监控平台、计费系统都需要应对 10-20 倍量级的流量增长。数据平台的弹性伸缩和实时处理能力成为关键。

2. 券商：Agent 模式下 Token 消耗结构性变化，AI 云服务进入涨价周期

📌 企鹅号 / 港股概念追踪（2026-03-11）
腾讯云涨价叠加 OpenClaw Token 消耗暴增，AI 云服务正式进入涨价周期。涨价逻辑：(1) 免费补贴不可持续；(2) Agent 对高质量推理的持续需求；(3) GPU 算力供给仍紧张。Agent 模式使 AI 算力消耗从"按需对话"转变为"全天候持续运行"，单用户 Token 消耗提升 2-3 个数量级。
📊 数据平台映射：推理成本上升将倒逼企业更关注 Token 使用效率和成本优化。数据平台需提供推理成本归因、缓存复用、智能路由等 FinOps 能力。
