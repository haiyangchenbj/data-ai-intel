Top Signals

1. Confluent Q2 2026 发布集：dbt adapter for Flink GA + AI 开发者工具全面可用

来源：Confluent Press Release / Confluent Blog

Confluent 在 Current London 大会上宣布多项 GA：开源 dbt adapter 将 Flink SQL 纳入 dbt 工作流，Managed MCP Server + Agent Skills 让 AI 编码助手直接操作流平台，Streaming Agents 和 Real-Time Context Engine 正式可用。同步发布 Confluent Private Cloud（CPC），基准测试显示同等负载下 broker 数量减少 73%。

💡 dbt 成为横跨批处理（Snowflake/Databricks/BigQuery）和流处理（Flink）的统一编排框架，企业数据工程团队可以用同一套 CI/CD 管理全部管道。MCP Server 的 GA 标志着 AI Agent 直接操作流基础设施从概念走向生产。

2. Snowflake dbt Projects 重大更新：列级血缘 + dbt Fusion 引擎 + Cortex Code

来源：Snowflake Release Notes

05-19 发布的更新包括：DAG 可视化新增列级血缘（基于 Horizon Catalog 元数据）、集成 Rust 重写的 dbt Fusion 引擎（无额外许可费）、多版本 dbt 支持（Core 1.9.4/1.10.15/Fusion 2.0.0-preview）、Cortex Code 全生命周期支持 dbt 工作流（生成/测试/调试/文档）。

💡 Snowflake 将 dbt 的编译和执行运行时深度嵌入平台，列级血缘让数据治理颗粒度达到字段级别，Fusion 引擎的免费内置消除了企业采用高性能 dbt 运行时的成本门槛。

3. 腾讯云正式发布大数据智能体工作台 DataBuddy

来源：IT168 / 钛媒体

05-19 正式发布。基于 WorkBuddy 同源 Agent 底层能力，用户通过自然语言对话即可完成数据接入、开发、治理、分析全链路任务。搭载于 Wedata 数据智能平台，原生连接 DLC 数据湖计算引擎。采用分层 Agent 协作架构（Harness AI），将 1-2 周数仓建设周期压缩至小时级。

💡 国内云厂商首个以 Agent 原生模式交付的全链路数据工作台，从"UI 驱动多页面操作"转向"对话驱动全链路自动化"，代表了数据平台交互范式的代际变化。

Product & Tech

1. Teradata 发布 Teradata Factory：融合 EDW/Lakehouse/AI 的私有部署一体机

来源：PR Newswire via Yahoo Finance

05-19 宣布，基于 Dell Technologies 企业计算和存储构建，将完整 Teradata 软件栈（含 AI Studio）整合到单一管理面，支持 EDW、Lakehouse 和 AI 工作负载。支持 Apache Iceberg、Delta Lake 和 S3 兼容对象存储。预计 Q3 2026 可用。

💡 在公有云经济性受到 GPU 推理和持续计算挑战的背景下，Teradata 押注"数据平台 + AI 平台一体化私有部署"，瞄准受监管行业的数据主权需求。

2. Confluent Private Cloud 发布：73% 更少 Broker + 集中策略治理

来源：Confluent Blog

Confluent Private Cloud（CPC）是面向私有基础设施的新软件包，提供增强型 broker（最高 10x 吞吐提升）和新 Gateway（网络隔离 + 集中策略执行）。基准测试证实同等负载下 broker 数量减少 73%，大幅降低 TCO。

💡 流处理的私有部署选项成熟度大幅提升，对于需要数据主权但又想获得云级敏捷性的组织而言，CPC 填补了 Confluent Platform 与 Confluent Cloud 之间的能力鸿沟。

Views & Research

1. Omdia：2025 中国 AI 云市场 567 亿元，首次定义 AI IaaS + MaaS 双轨框架

来源：每经/格隆汇

Omdia 05-19 发布《中国AI云市场份额2025》，将 AI 云首次系统拆分为 AI IaaS（占 69%）和 MaaS（占 31%，含 MPS 模型生产+MCS 模型消费）。阿里云以 38.1% 份额居首，火山引擎 20.4% 居二，百度云/腾讯云/天翼云列三至五。市场总规模达 567 亿元人民币。

💡 首个将 MaaS 进一步拆为"模型生产"和"模型消费"的市场框架。对数据平台厂商意味着：数据平台与 AI IaaS 的集成深度（训练数据管理、特征工程、推理数据流）直接决定在 AI 云市场的定位。
