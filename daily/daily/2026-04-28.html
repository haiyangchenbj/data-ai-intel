1. 国家数据局：2026 年"数据要素×"大赛正式发布，首次将"数据基础设施"设为独立专业赛道

来源：国家数据局 — 关于举办2026年"数据要素×"大赛的通知（国数政策〔2026〕18号）

国家数据局于 04-27 正式发布通知，联合 21 个部委举办 2026 年"数据要素×"大赛（主题：数据赋能 乘数而上）。大赛设 16 个行业领域赛道和 1 个专业赛道，覆盖 121 个赛题方向。地方分赛已启动，全国总决赛拟 9 月举行。值得关注的是，"数据基础设施"被首次设为独立专业赛道（第 17 赛道），包含 7 个赛题方向：原生应用场景赋能、技术应用创新、运营机制探索、多源异构算力资源池化、算力智能感知与监测、算力精准匹配与调度、绿色算电融合。

映射到数据平台的判断：这是"数据要素×"三年行动计划（2024-2026）收官年的关键动作。数据基础设施首次从通用技术支撑角色上升为独立赛道，信号明确：政策层面已将数据基础设施视为独立产业生态而非附属于某个行业的技术工具。金融服务赛道中的"隐私计算协同应用""可信数据空间"等赛题方向也与数据平台的联邦查询、数据治理能力直接相关。

融资Oracle 密歇根数据中心园区 $160 亿融资落地——Stargate 项目持续推进

来源：Data Center Dynamics / Bank of America Newsroom

Related Digital 与 Blackstone 于 04-27 宣布，为密歇根州 Saline Township 的 Oracle 数据中心园区完成 $160 亿融资。Pimco 购入 $100 亿债券（2045 年到期，票面利率 7.5%），Blackstone 以 $20 亿参与股权投资，Bank of America 承销 $140 亿债券。园区容量 1GW，为 OpenAI Stargate 项目的一部分，服务于 Oracle Cloud Infrastructure。大规模建设已全面展开。

对数据平台的影响：单个数据中心项目 $160 亿融资规模再次刷新行业记录。虽然项目主要服务 AI 训练基础设施，但 Oracle Cloud Infrastructure 是直接受益方——基础设施扩张将直接扩大其云数据平台的计算容量和客户承载力。AI 基础设施的资本密度已远超数据平台独立赛道的融资规模，资本流向的虹吸效应持续加剧。

跟踪Microsoft-OpenAI 合作协议修订：OpenAI 可通过任意云提供商服务客户

来源：Microsoft Official Blog — The next phase of the Microsoft-OpenAI partnership

Microsoft 于 04-27 宣布与 OpenAI 修订合作协议。核心变化：（1）Microsoft 仍是主要云合作伙伴，OpenAI 产品优先在 Azure 发布，但 OpenAI 现在可通过任意云提供商向客户提供所有产品；（2）Microsoft 对 OpenAI 模型的 IP 许可延续至 2032 年但变为非独占；（3）Microsoft 不再向 OpenAI 支付收入分成，OpenAI 向 Microsoft 的收入分成持续至 2030 年并设总额上限。对数据平台而言，OpenAI 多云服务条款的松绑意味着 Azure 在 AI+Data 场景中的排他优势被削弱——企业使用 OpenAI 能力时不再被绑定 Azure 数据栈。

等待确认信号：OpenAI 与 Google Cloud / AWS 的具体合作落地方式；Azure AI+Data 生态的客户留存率变化；多云 OpenAI 对 Fabric/Azure Databricks 等数据平台方案的实际影响。

跟踪Google Cloud Next '26 公告持续进入落地验证期

来源：综合 Google Cloud Next '26 大会公告 + Futurum 分析

Google Cloud Next 大会已闭幕一周（04-22~24），核心数据平台公告（Agentic Data Cloud / Cross-Cloud Lakehouse / Knowledge Catalog / Data Agent Kit / Data Engineering Agent GA）进入落地验证期。跨云 Lakehouse 和 Knowledge Catalog 仍为 Preview 状态。同期 Microsoft-OpenAI 多云条款松绑，为 Google Cloud 争取 OpenAI 工作负载提供了窗口。

来源：综合 Databricks / Snowflake / Google Cloud 近期发布

三大平台 Agent 架构路线持续分化——Google 走"跨云推理平面"、Databricks 走"开放治理控制平面"（Supervisor Agent SDK 上周进入 Beta）、Snowflake 走"平台内聚控制平面"。OpenAI 多云条款松绑为竞争格局增加新变量。06 月 Databricks Data+AI Summit 和 Snowflake Summit 将是集中回应 Google 并争取 OpenAI 多云机遇的关键窗口。
