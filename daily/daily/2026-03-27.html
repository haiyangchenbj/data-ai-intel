Top Signals

1. Databricks 12.2 LTS / Amazon DocumentDB 3.6 / Azure PostgreSQL 11-13 本月集中退役——多平台 EOL 叠加倒逼升级

来源：Datavail - Data Platforms Reaching End of Life in March 2026（March 26, 2026）

Datavail 发布3月数据平台 EOL 综合报告，三大平台版本同期退役：（1）Databricks 12.2 LTS 完全退役（无延期支持），17.2 非 LTS 版本同步停止支持，建议升级至 13.3+ 或 17.3 LTS；（2）Amazon DocumentDB 3.6 月末 EOL，3月30日前未升级将自动进入付费延期支持，建议就地升级至 DocumentDB 5.0；（3）Azure Database for PostgreSQL 11/12/13 已 EOL，免费延期支持持续至7月31日，之后转入最长3年付费延期，建议升级至 PostgreSQL 17 或 18。此外预告4月 MySQL 8.0 和 IBM Db2 11.1 LUW 即将退役。

为什么对数据平台重要：多个主流版本同时退役形成"升级压力叠加"：Databricks 用户必须跨越至少一个 LTS 大版本，DocumentDB 用户面临3月30日自动付费的倒计时，PostgreSQL 用户需要跨多个主版本升级。对于同时使用多平台的企业，这些 EOL 日期的密集重叠可能引发集中的迁移评估和技术栈简化决策——这正是湖仓平台厂商争取客户整合的机会窗口。4月 MySQL 8.0 和 Db2 11.1 的接续退役意味着这波升级压力将持续至少两个月。
