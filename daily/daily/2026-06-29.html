【降级】Snowflake 完成 Delta Direct 生态扩展：Dynamic Tables 与 Insert-Only Streams 正式支持 Apache Iceberg 表

来源： Snowflake Release Notes
为什么值得继续看： Snowflake Release 10.22（6/20-25 部署）为 Delta Direct 表（Snowflake 管理的 Iceberg 表）新增了 Dynamic Tables 和 Insert-Only Streams 两项 GA 能力。Dynamic Tables 支持在 Iceberg 表上实现声明式增量物化刷新，Insert-Only Streams 则为 CDC 场景提供变更捕获。这是 Snowflake 在开放表格式互操作方向上持续推进的重要一步——Iceberg 表正从"可读取"走向"可完整操作"，但发布日期略早于本期窗口，降级收录。
