降级 2. Apache Flink Agents 0.2.1 发布，MCP 兼容性和稳定性改善

来源：Apache Flink Blog — Flink Agents 0.2.1 Release Announcement
为什么值得继续看：Flink Agents 0.2.1（03-26 发布）修复了远程 MCP 服务器连接兼容性问题和 Jackson 反序列化 bug，新增日志美化输出并优化包体积。作为将 Flink 流处理能力与 AI Agent 编排相结合的子项目，其 MCP 集成可靠性直接影响 Agent 在数据管道中的生产可用性。因发布日期略早于窗口起点而降级收录。
