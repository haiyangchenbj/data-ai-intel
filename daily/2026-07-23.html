Progress Software $400M收购Domo AI与数据平台业务

来源： Progress Software Press Release
摘要： Progress Software（NASDAQ: PRGS）于7月22日宣布以$400M现金收购Domo（NASDAQ: DOMO）的AI与数据平台业务，包括其全部资产、员工、客户合同和知识产权。Domo拥有超过2,400家企业客户，平台覆盖数据集成、治理分析、自动化和AI驱动的数据产品。交易为资产收购而非合并，预计在Progress 2026财年内（11月30日前）完成交割。Domo此前因年化经常性收入下降触发贷款违约条款，董事会自春季启动战略审查。
为什么对数据平台重要： 这是Progress自2023年收购MarkLogic后的第三次数据平台收购，表明传统基础设施软件厂商正通过并购系统性地构建AI数据平台产品栈。Domo在面临财务压力时被收购，反映了独立BI/数据分析平台在AI时代的生存困境——要么自建AI能力，要么被拥有更完整产品栈的厂商整合。$400M对价（约Domo年化营收5x）也提供了当前环境下成熟数据分析平台的估值锚点。

腾讯云发布ADP 4.0海外版与企业级CodeBuddy IDE

来源： 腾讯云开发者社区
摘要： 在2026世界人工智能大会（WAIC）期间，腾讯云于7月22日正式发布企业级智能体开发平台ADP 4.0海外版，同步推出CodeBuddy IDE（全流程AI一体化开发工具）。ADP 4.0海外版围绕触达、交互、生态、连接四大维度升级，接入LINE、Telegram等海外主流社交渠道，内置多家国际大模型本地化适配。CodeBuddy IDE定位为覆盖编码、调试、部署全流程的AI开发环境。此外腾讯云还首次系统性展示具身智能全栈布局，WorkBuddy独立App同步上线iOS/Android/鸿蒙。
为什么对数据平台重要： ADP 4.0海外版发布标志着中国企业级Agent开发平台从国内市场走向全球化竞争，其"Claw模式"（Agent与Workflow双向互调）代表了Agent落地企业的工程化方向——开放场景理解由Agent承担，标准化流程由工作流保证严谨。CodeBuddy IDE的意义在于将AI编程从辅助代码补全升级为覆盖数据工程全流程的一体化开发环境，直接影响数据工程师的工具链选型。

阿里云EventHouse事件湖仓正式启动商业化

来源： 阿里云公告
摘要： 阿里云EventBridge旗下EventHouse（事件仓）于2026年7月23日00:00正式结束免费公测并启动商业化收费。EventHouse于2026年2月首次发布，定位为新一代云原生事件湖仓，将高吞吐消息队列、结构化数据库和非结构化对象存储统一抽象为标准事件数据模型，解决事件的"存储、治理与智能分析"问题。
为什么对数据平台重要： EventHouse商业化的核心信号是"事件湖仓"作为独立产品品类从概念验证走向企业采购清单。这意味着企业对实时事件数据的持久化存储、Schema治理和智能分析的需求已从自有搭建转向采购商业化云服务。EventHouse借鉴Lakehouse架构理念将事件数据纳入统一湖仓治理体系，也代表了流批一体架构中最薄弱层——事件存储层——正在被云厂商产品化填补。

Snowflake SHOW CORTEX BASE MODELS 命令正式GA

来源： Snowflake Release Notes
摘要： Snowflake于7月23日将SHOW CORTEX BASE MODELS SQL命令正式GA。该命令可列出账户内所有可用的Cortex基础模型，包括生命周期状态（GA/PUPR/PRPR/LEGACY/EOL）、可用区域、Legacy日期和EOL日期。配合RBAC机制，账户管理员可以通过SNOWFLAKE.MODELS.CORTEX_BASE_MODELS_REFRESH存储过程刷新模型对象，并通过GRANT APPLICATION ROLE精细控制不同角色对特定模型的访问权限。
对数据平台的影响： 该命令的GA标志着Snowflake将AI模型治理正式纳入其数据平台RBAC体系。企业现在可以像管理数据库表权限一样管理大模型访问——哪些部门可用哪些模型、何时需要迁移至替代模型，全部可编程、可审计。这是"数据治理"边界向AI模型层延伸的又一个里程碑，也为在Snowflake上构建AI应用的企业提供了合规采购的基础设施。
