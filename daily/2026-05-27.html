Databricks Genie Code Agent 模式默认启用（合规工作区）

来源：Databricks Release Notes · 2026-05-25 ｜ 信息置信度 Level A

Genie Code 的 Agent 模式现已对启用合规安全配置（Compliance Security Profile）的工作区默认可用，覆盖 HIPAA、PCI-DSS 和 FedRAMP Moderate 控制标准的环境。Agent 模式核心能力：

  • 从单个自然语言提示出发，自动执行多步操作
  • 自动检索相关数据资产
  • 生成并运行代码
  • 自动修复错误
  • 输出可视化结果
So What：数据分析从"一问一答"升级为"一问多步自动完成"，且率先在监管敏感行业（医疗/金融/政府）落地。Databricks 将 Agent 模式视为企业级标准能力而非实验性功能，对同类产品形成定义性压力。
