阿里云 EMR AI 助手正式发布：大数据运维进入全栈 AIOps 时代

来源：阿里云开发者社区
摘要： 阿里云于7月21日正式发布 EMR AI 助手，面向大数据运维场景提供 AIOps 智能运维能力。产品直连集群，基于大模型实现智能诊断、全面巡检、主动推送和专家技能四大核心能力，涵盖 AI 日报自动生成、慢 SQL 根因分析、集群健康巡检、IM 通道告警推送等场景。与通用大模型的区别在于：EMR AI 助手能实时读取集群状态、关联多维指标形成因果链推理、自动编排诊断-优化-验证全流程。目前已支持 EMR Serverless StarRocks，后续将扩展至更多引擎。
为什么对数据平台重要： 这是国内主流云厂商首次将 AIOps 以产品化形态嵌入大数据平台，而非停留在"问答助手"层面。AI 日报从 50 个监控指标中自动提炼 3 项需关注事项并附带根因推理和处方建议，将大数据运维从"人工多界面切换排查"升级为"AI 主治医生"模式。对数据平台工程效率的影响直接：厂商声称可提升 10 倍运维效率。更重要的是，它标志着大数据平台的运维范式正在从规则驱动走向 AI 驱动。

阿里云 Dataphin V6.2 发布：数据治理平台原生内嵌向量检索与 Agent API

来源：阿里云帮助中心
摘要： 阿里云智能数据建设与治理平台 Dataphin 于7月21日发布 V6.2 版本。核心更新包括：新增数据集 API，原生支持向量检索和文件下载链接生成，为 AI Agent 提供开箱即用的知识查询服务；新增支持 Milvus 2.6.x 向量数据库作为离线集成数据源；离线物理表新增支持 StarRocks、Doris、SelectDB；数据库 SQL 新增支持 TDSQL、ClickHouse 等；数据质量规则新增超时自动终止与失败重试配置。
为什么对数据平台重要： Dataphin V6.2 的核心信号是数据治理平台正在从"管数据"升级为"管 AI 入口"。数据集 API 原生支持向量检索意味着企业不需要单独搭建向量数据库即可让 Agent 调用治理后的企业数据——Data Governance 与 AI Access 正在融合为一个平台能力。新增 Milvus 数据源和多款 OLAP 引擎支持也表明，国内数据治理平台正在加速兼容开放生态，降低企业对单一技术栈的锁定风险。

Snowflake CoCo Desktop 正式 GA：数据平台开发工具迈入 Agentic IDE 时代

来源：Snowflake Documentation
摘要： Snowflake 于7月21日宣布 CoCo Desktop 正式 GA。CoCo Desktop 是一款原生 Mac/Windows 桌面应用，在 IDE 风格的界面中集成了文件编辑器、终端、Agentic AI Loop、Agentic Browser、Agentic Notebook 以及深度 Snowflake 感知能力。用户可在受治理的环境中通过自然语言完成数据探索、开发和管理操作，无需离开 Snowflake 的安全边界。
对数据平台的影响： CoCo Desktop GA 代表数据平台工具链的交互范式正在从"Web 控制台"向"Agentic IDE"迁移。与 Databricks 的 Genie/Agent Bricks 方向一致，Snowflake 选择将 Agent 能力嵌入一个桌面端全功能 IDE 而非 Web 界面，这一选择反映了数据平台对"开发者体验"的重视——Agent 不只服务于业务用户的分析问答，更要服务于数据工程师的开发运维全流程。但需注意 CoCo Desktop 依赖于 Snowflake 的 Cortex Code 模型服务，存在一定平台锁定风险。

Databricks 仓库级语句超时与工作空间访问控制 GA：运维精细化和安全合规补全

来源：Azure Databricks Release Notes
摘要： Databricks 于7月21-22日落地多项功能更新：SQL 仓库级语句超时（Beta，7/21）允许不同仓库设置独立查询超时；Context-based ingress control for workspaces 正式 GA（7/21），支持基于身份、网络来源和请求类型的零信任访问策略；Databricks 内置示例数据集通过 OpenSharing 协议在 Unity Catalog 的 samples catalog 中开放访问（7/22）。
对数据平台的影响： 三项更新虽属增量功能，但各自指向不同方向：仓库级超时提升了多租户 SQL 环境的运维精细度；基于上下文的入口控制补全了企业级零信任安全架构的关键缺口；示例数据集通过 OpenSharing 分发则是开放共享协议在平台内部的首次落地验证。三个方向（运维精细化、安全合规、开放共享）共同构成了平台成熟度提升的信号。
